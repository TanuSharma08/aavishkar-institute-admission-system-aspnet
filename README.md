# 👩‍💻 About The Project

The **Aavishkar Institute Management System (ASP.NET Version)** is a complete web-based institute management platform developed using ASP.NET WebForms and C#.

The system helps manage:
- Admissions
- Courses
- Student communication
- Leave applications
- Feedback
- Announcements
- Study material distribution

This version is an enhanced migration of the earlier PHP-based system with improved project organization, SQL Server integration, and additional functionalities such as online study material management.

---

# ⚙️ Setup Instructions

## Prerequisites

- Visual Studio 2019 or later
- SQL Server LocalDB / SQL Server Express
- .NET Framework 4.5+

---

## 1️⃣ Open in Visual Studio

- Open the project folder
- Open any `.aspx` file
- The solution will load automatically

---

## 2️⃣ Configure Database

The project uses SQL Server LocalDB with `.mdf` files stored inside the `App_Data/` folder.

Database name:

```text
aavishkar_institute
```

Update the connection string inside:

```text
Web.config
```

---

## 3️⃣ Run the Application

Press `F5` in Visual Studio

or open:

```text
http://localhost:xxxxx/Default.aspx
```
